À propos de ce projet

Ce projet est un serveur HTTP développé dans le cadre du TP3 d'ASRIR.

Objectifs

  • Implémenter un serveur HTTP multi-thread
  • Gérer plusieurs clients en parallèle
  • Servir des pages web statiques
  • Gérer les exceptions et erreurs

Technologies utilisées

  • Java pour le serveur
  • Sockets TCP
  • Threads pour le parallélisme
  • HTML/CSS pour les pages web
